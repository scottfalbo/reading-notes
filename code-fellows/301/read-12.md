# 301 Read-12: Partials

## EJS Partials
+ Partials are for reusing the same HTML across multiple views.
  + `head`, `headers`, `footers`
+ Partials snippets are saved in their own *.ejs file and then called into pages.
  + `<%-include('filepath/file')%>`
